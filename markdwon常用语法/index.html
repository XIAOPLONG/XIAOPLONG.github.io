# markdown 常用

# 标题语法

  • 相应标题前加对应个 #，# 数代表了标题的级别。（注意 #后有个空格）
  • 或者在文本的下方添加任意数量的 = 标识一级标题，或者 - 标识二级标题

# 段落语法

  • 用空白的一行或者多行文本进行分隔

不要用空格或者制表符缩进段落

# 换行语法

  • 结尾空格方式

输入两个或者多个空格后进行回车换行。

  • HTML 的 "br" 标签

或者直接用 "br" 标签换行

# 强调语法

  • 粗体强调

在单词或者短语的前后各添加两个星号

  • 斜体

前后各添加一个星号

  • 粗体和斜体

前后各添加三个星号

# 引用语法

  1. 创建块引用

段落前 ">" 符号

  1. 多个段落的块引用

块引用可以包含多个段落，为段落之间的空白行添加一个 ">" 符号

  1. 嵌套块引用

在要嵌套的段落前添加一个 ">>" 符号

  1. 带有其它元素的块引用

"> - 这是一个带其它元素的块引用"

显示效果如下：

  • 这是一个带其它元素的块引用

# 分隔线语法

单独一行上使用三个或者多个星号 (***), 破折号 (---), 或者下划线 (___), 并且不能包含其它内容。

为了兼容性，分割线前后均要添加空白行。

# 链接语法

[超链接显示名](超链接地址 "超链接title")

用尖括号可以很方便的将 URL 或者 Email 地址变为可点击的链接。

# 图片语法

![图片alt](图片链接 "图片title")
